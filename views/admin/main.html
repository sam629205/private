系统环境检测：
主机名 (IP：端口)： echo $_SERVER['SERVER_NAME'].' ('.$_SERVER['SERVER_ADDR'].':'.$_SERVER['SERVER_PORT'].')'
程序目录： {&web_path}
Web服务器： echo $_SERVER['SERVER_SOFTWARE']
PHP 运行方式： echo PHP_SAPI
PHP版本： echo PHP_VERSION
MySQL 版本： if (function_exists("mysql_close")) { echo mysql_get_client_info(); }else{ echo '不支持'; }  
GD库版本： if(function_exists('gd_info')){ $gd = gd_info(); echo $gd['GD Version']; }else{ echo "不支持"; }
最大上传限制： if (ini_get('file_uploads')) { echo ini_get('upload_max_filesize'); }else{ echo 'Disabled'; }
最大执行时间： echo ini_get('max_execution_time')秒
采集函数检测： if (ini_get('allow_url_fopen')) { echo '支持'; }else{ echo '不支持'; }
